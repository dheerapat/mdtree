# Cognitive Load Theory

An introduction to how working memory affects learning.

## What is Cognitive Load?

Cognitive load refers to the mental effort being used in the working memory.
When the cognitive load is too high, learning is impaired.

### Types of Cognitive Load

There are three main types of cognitive load.

#### Intrinsic Load

Intrinsic load is the inherent difficulty of the material itself.
It is determined by the complexity of the content and the learner's prior knowledge.

#### Extraneous Load

Extraneous load is caused by poor instructional design.
It does not contribute to learning and should be minimised.

#### Germane Load

Germane load is the mental effort devoted to processing and forming schemas.
It contributes positively to long-term memory.

## Managing Cognitive Load

Instructional designers can apply several strategies to manage cognitive load effectively.

### Chunking

Breaking information into smaller units reduces intrinsic load.
Each chunk should be meaningful and self-contained.

### Worked Examples

Providing fully worked examples reduces extraneous load for novice learners.
As expertise grows, the guidance can be faded gradually.

### Dual Coding

Combining verbal and visual information leverages both verbal and visuospatial channels.
This can increase the effective capacity of working memory.

## Implications for eLearning

Online learning environments must be carefully designed to minimise unnecessary cognitive load.

### Screen Design

Avoid cluttered interfaces. Use whitespace, clear typography, and logical layout.

### Multimedia Principles

Follow Mayer's multimedia principles: coherence, signalling, redundancy, contiguity, and modality.
